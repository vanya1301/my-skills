# Personal Skills

Collection of custom OpenCode skills.

## Installation

### One-line install (Recommended)

```bash
curl -fsSL https://github.com/vanya1301/my-skills/releases/latest/download/install.sh | bash
```

### Manual installation

```bash
INSTALL_DIR=~/.config/opencode/skills/personal
curl -L https://github.com/vanya1301/my-skills/releases/latest/download/skills.tar.gz | tar -xz -C "$INSTALL_DIR" --strip-components=1
```

## Skills

- **lazy-senior** - Makes Claude code like a lazy senior engineer (pragmatic, minimal, ship-it mentality)
